UX Design Notes





Consistency:

  • Builds trust with the user
  • Every interaction helps connect the user with the company

Seamless:

  • Users should be able to pick up where they left off across devices
  • Important to consider the path a user takes to complete a task

Availability:

  • The appropriate access for tasks should be provided
  • Conducting research with people to understand how they interact with a platform

Context Specific:

  • Understand the context of use and the users' journey
  • Incorporate helpful and useful items for navigating
